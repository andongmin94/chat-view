# 치지직 설정

치지직(네이버) 방송의 채팅을 챗뷰 오버레이로 띄우는 방법을 안내해 드리겠습니다.

## 지원하는 URL 형식

```
https://chzzk.naver.com/chat/<채널ID>
```

방송 페이지에서 채팅 전용 탭이나 팝업을 열면, 위 패턴으로 시작하는 주소를 얻을 수 있습니다.

## 채팅 URL 찾는 방법

1. 브라우저에서 치지직 방송 페이지에 접속해 주세요.
2. 채팅 영역을 새 창으로 여는 기능이 있다면 사용하거나, 개발자 도구에서 iframe src를 확인해 보세요.
3. 주소가 `https://chzzk.naver.com/chat/`로 시작하는지 확인한 후 복사해 주세요.
4. 챗뷰의 URL 입력 모달에 붙여넣고 **적용**을 눌러주세요.

## 고정 모드 활용하기

| 상황 | 권장 설정 |
|------|------------|
| 방송 중 게임/작업에 집중 | 고정 ON — 클릭이 밑으로 통과되어 방해되지 않습니다 |
| 채팅 스크롤이나 복사가 필요할 때 | 고정 OFF — 채팅과 직접 상호작용할 수 있습니다 |

## 글꼴 및 크기

현재 버전에서는 치지직 기본 채팅 UI가 그대로 로드되기 때문에, 별도 폰트나 크기 옵션은 없습니다.

앞으로 커스텀 CSS 기능이 추가되면 스타일도 자유롭게 바꿀 수 있을 것입니다.

## 문제가 생겼을 때

| 증상 | 이렇게 해 보세요 |
|------|-------------------|
| URL 오류 | `https://chzzk.naver.com/chat/`로 시작하는지 다시 확인해 보세요 |
| 로드가 느립니다 | 네트워크 상태를 확인하고, 잠시 대기 후 다시 적용해 보세요 |
| 채팅 입력이나 선택이 안 됩니다 | 고정 모드가 ON인지 확인해 보세요. OFF로 전환하면 상호작용 가능합니다 |

::: tip 방송 시작 전에 세팅해 두세요
오버레이 위치와 크기를 미리 조정해 놓고, `고정 ON`으로 전환해 두면 방송 중 수정할 일이 줄어듭니다.
:::
