# 챗뷰 시작하기

챗뷰는 스트리밍/방송 환경에서 채팅 화면을 별도 오버레이 창으로 띄우는 Tauri 기반 데스크톱 앱입니다.

## 빠른 링크
- [위플랩 설정](./weflab.md)
- [치지직 설정](./chzzk.md)
- (예정) [숲 설정](./soop.md)

## 설치 및 실행
1. GitHub 릴리즈에서 패키지를 내려받습니다.
2. Windows에서는 릴리즈의 `.exe` 파일을 실행합니다.
3. 또는 소스에서 직접 실행합니다.

```bash
cd packages
npm install
npm run tauri:dev
```

## 지원 URL 형식
현재 아래 URL 형식을 지원합니다.

```text
https://weflab.com/page/...
https://chzzk.naver.com/chat/...
```

## 기본 사용 흐름
1. 메인 창에서 URL을 입력합니다.
2. 적용하면 투명 오버레이 창이 생성됩니다.
3. `고정 ON`이면 항상 위 + 클릭 스루(완전 투명) 상태가 됩니다.
4. `고정 OFF`이면 반투명 배경으로 바뀌고, 오버레이 창 전체를 드래그해 이동할 수 있습니다.
5. 위치/크기/고정 상태/URL은 자동 저장되어 다음 실행 때 복원됩니다.

## 자주 묻는 문제
| 증상 | 해결 방법 |
|------|-----------|
| URL 입력해도 오류 | 지원 URL 형식인지 확인 (`weflab`, `chzzk`) |
| 오버레이가 안 뜸 | URL 재적용 또는 리셋 후 재시도 |
| 드래그가 안 됨 | `고정 OFF` 상태인지 확인 |
| 위치 복원이 이상함 | 설정 리셋 또는 저장 파일 삭제 후 재설정 |

## 저장 데이터
설정은 앱 데이터 디렉터리의 `store.json`에 저장됩니다.

저장 키:
- `chatUrl`
- `overlayFixed`
- `overlayWindowBounds`

## 업데이트
현재는 앱 내에서 최신 릴리즈를 확인하고 다운로드 링크를 안내하는 방식입니다.
자동 설치형 업데이터는 아직 적용되지 않았습니다.
