# 챗뷰 시작하기

스트리밍이나 방송을 하다 보면, 채팅 창을 항상 눈에 띄워두고 싶을 때가 있습니다.

챗뷰는 위플랩이나 치지직 채팅을 투명한 오버레이 창으로 띄워주는 앱입니다.

게임이나 다른 프로그램 위에 채팅을 겹쳐 놓고, 필요할 때만 잠깐 보면 됩니다.

## 📖 가이드 목차
- [위플랩 설정](./weflab.md) — 위플랩 채팅을 오버레이로 띄우는 방법
- [치지직 설정](./chzzk.md) — 치지직 채팅을 오버레이로 띄우는 방법
- (예정) [숲 설정](./soop.md) — SOOP 플랫폼 지원을 준비하고 있습니다

## 설치 및 실행

GitHub 릴리즈에서 `.exe` 파일을 받아 바로 실행하면 됩니다. 별도 설치 과정이 없습니다.

소스 코드에서 직접 실행하고 싶다면 아래 명령어를 따라 해 주세요.

```bash
cd packages
npm install
npm run tauri:dev
```

## 지원하는 URL

현재 아래 플랫폼의 채팅 URL을 지원합니다.

```text
https://weflab.com/page/...
https://chzzk.naver.com/chat/...
```

## 기본 사용 방법

1. **URL 입력** — 메인 창에서 채팅 URL을 붙여넣고 적용을 눌러주세요.
2. **오버레이 확인** — 투명한 오버레이 창이 화면에 나타납니다.
3. **고정 모드 선택** — `고정 ON`이면 채팅이 항상 위에 붙고 클릭이 밑으로 통과됩니다. `고정 OFF`로 바꾸면 창을 드래그해서 이동할 수 있습니다.
4. **자동 저장** — 위치, 크기, 고정 상태, URL이 모두 저장되어 다음 실행 때 그대로 복원됩니다.

::: tip 고정 모드 활용 팁
방송 중에는 `고정 ON`으로 두면 게임이나 다른 프로그램을 클릭할 때 채팅 창이 방해되지 않습니다. 채팅을 스크롤하거나 복사해야 할 때만 잠시 `고정 OFF`로 전환하면 됩니다.
:::

## 문제가 생겼을 때

| 증상 | 이렇게 해 보세요 |
|------|------------------|
| URL을 입력했는데 오류가 납니다 | 지원하는 URL 형식(`weflab`, `chzzk`)인지 확인해 보세요 |
| 오버레이가 안 뜹니다 | URL을 다시 적용하거나, 리셋 후 재시도해 보세요 |
| 드래그가 안 됩니다 | `고정 OFF` 상태인지 확인해 보세요 |
| 위치가 이상하게 복원됩니다 | 설정 리셋 후 다시 위치를 잡아 보세요 |

## 데이터 저장 위치

설정은 앱 데이터 디렉터리의 `store.json`에 저장됩니다. 채팅 URL, 고정 상태, 오버레이 창의 위치와 크기가 모두 여기에 기록됩니다.

## 업데이트

현재는 앱 안에서 최신 릴리즈를 확인하고 다운로드 링크를 안내해 주는 방식입니다. 자동 업데이트 기능은 앞으로 준비할 계획입니다.
