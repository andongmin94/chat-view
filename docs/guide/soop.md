# 숲(SOOP) 설정

숲(SOOP)은 많은 스트리머분들이 사용하시는 라이브 방송 플랫폼입니다.

챗뷰에서 숲 채팅을 오버레이로 띄울 수 있다면, 방송 화면에 시청자 반응을 자연스럽게 보여줄 수 있어서 정말 편리하겠지요?

이 문서에서는 숲 채팅을 챗뷰와 연동하는 방법을 안내해 드립니다.

::: warning 아직 지원되지 않는 기능입니다
숲(SOOP) 연동은 현재 개발 중이며, 아직 정식으로 지원되지 않습니다. 지금은 챗뷰의 URL 허용 목록에 숲이 포함되어 있지 않기 때문에, 숲 채팅 주소를 입력해도 동작하지 않습니다. 정식 지원이 시작되면 이 문서도 함께 업데이트할 예정입니다!
:::

## 현재 지원 현황

챗뷰는 채팅 URL을 입력받을 때, 안전을 위해 허용된 도메인만 통과시키는 검증 과정을 거칩니다.

현재 허용 목록에는 **위플랩(weflab)** 과 **치지직(chzzk)** 만 등록되어 있습니다.

숲이 이 목록에 추가되면, 다른 플랫폼과 동일한 방식으로 바로 사용하실 수 있게 됩니다.

## 예상 사용 방법

정식 지원이 되면, 아래와 같은 흐름으로 사용하실 수 있을 것입니다.

1. **채팅 팝업 주소 확인하기** — 숲 방송 페이지에서 채팅 팝업 또는 iframe 주소를 찾아주세요.
2. **챗뷰에 URL 입력하기** — 메인 제어 창에서 확인한 주소를 붙여넣고 적용 버튼을 눌러주세요.
3. **오버레이 배치하기** — 고정 모드를 켜면 방송 중에도 오버레이 위치가 흔들리지 않고 고정됩니다.

예상되는 URL 형태는 아래와 비슷할 것입니다 (실제 도메인은 달라질 수 있습니다).

```
https://soop.example.com/chat/<채널ID>
```

## 문제가 생길 수 있는 상황

정식 지원 전이라 아래와 같은 문제를 만나실 수 있습니다. 참고해 주세요!

| 증상 | 왜 그런가요? | 어떻게 하면 되나요? |
|------|-------------|-------------------|
| URL을 입력했는데 오류가 뜹니다 | 아직 숲 도메인이 허용 목록에 없기 때문입니다 | 정식 지원 업데이트를 기다려 주세요 |
| 주소는 통과했는데 화면이 비어 있습니다 | 숲 채팅이 로그인 인증이나 쿠키를 요구할 수 있습니다 | 브라우저에서 먼저 숲에 로그인한 뒤 주소를 다시 확인해 보세요 |

## 앞으로의 계획

- 숲 서비스의 실제 채팅 URL 구조를 파악해서 허용 패턴을 구체화할 예정입니다.
- 다른 플랫폼과 스타일이 충돌하지 않는지 테스트할 것입니다.
- 오버레이에서 폰트와 색상이 잘 보이는지 가독성도 꼼꼼히 확인할 예정입니다.
- 필요하다면 플랫폼별 커스텀 CSS를 적용할 수 있는 기능도 고려하고 있습니다.

정식 지원 소식이 있으면 이 문서를 통해 안내해 드리겠습니다. 조금만 기다려 주세요!
