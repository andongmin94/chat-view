# 위플랩 설정

위플랩(weflab)을 방송에서 사용하고 계신가요?

챗뷰를 사용하면 위플랩 채팅을 투명한 오버레이 창으로 화면 위에 띄울 수 있습니다.

방송 화면을 가리지 않으면서도 시청자 채팅을 실시간으로 확인할 수 있어서, 채팅 반응을 놓치지 않고 방송에 집중할 수 있습니다.

이 가이드에서는 위플랩 채팅 URL을 챗뷰에 연결하는 방법을 단계별로 안내해 드리겠습니다.

## 채팅 URL 가져오기

챗뷰에 위플랩 채팅을 연결하려면 먼저 올바른 URL이 필요합니다. 아래 순서대로 따라 해 주세요.

1. 브라우저에서 위플랩 스트림 페이지(또는 채팅이 포함된 페이지)에 접속해 주세요.
2. 주소창의 URL을 복사해 주세요. 이때 주소가 아래 형식인지 꼭 확인해 주세요.

```
https://weflab.com/page/<페이지ID>
```

3. 챗뷰의 URL 입력 창에 복사한 주소를 그대로 붙여넣고 **[적용]** 버튼을 눌러 주세요.

이게 전부입니다! 정상적으로 연결되면 위플랫 채팅이 투명 오버레이로 화면에 나타나게 됩니다.

::: tip 주소 형식 확인하기
URL이 반드시 `https://weflab.com/page/`로 시작해야 합니다. `/page/` 뒤에 오는 페이지 ID 부분은 수정 없이 그대로 사용하시면 됩니다.
:::

## 고정 모드 활용하기

오버레이 창의 위치를 잡았다면, **고정 모드**를 활용해 보세요.

고정 모드를 켜면 오버레이 창이 실수로 움직이는 걸 방지할 수 있고, 마우스 클릭이 오버레이를 통과(클릭 패스스루)하기 때문에 뒤에 있는 프로그램도 자유롭게 조작할 수 있습니다.

방송 중에는 **고정 모드를 켜 두는 것**을 추천합니다.

채팅 내용을 복사하거나 링크를 클릭하는 등 오버레이와 직접 상호작용이 필요한 순간에만 잠깐 고정을 꺼 주세요.

::: tip OBS와 함께 사용할 때
OBS 화면 위에 오버레이를 겹쳐서 사용하신다면, 위치를 정확히 맞춘 뒤에 고정 모드를 켜 두세요. 방송 중 실수로 오버레이를 건드려 위치가 틀어지는 걸 막을 수 있습니다.
:::

## 스타일 커스터마이징

위플랩 채팅의 글씨 색상이나 배경이 마음에 들지 않으신다면, 우선 위플랩 페이지 자체에서 제공하는 다크/라이트 모드 전환 기능이 있는지 확인해 보세요.

모드를 변경한 뒤 챗뷰에 URL을 다시 입력하면 바뀐 스타일이 적용됩니다.

현재 버전의 챗뷰에서는 커스텀 CSS 주입 기능을 아직 지원하지 않지만, 향후 업데이트에서 추가될 예정입니다. 

그때까지는 브라우저 확장 프로그램(CSS 커스터마이저 등)으로 원하는 스타일 규칙을 미리 정리해 두시면 나중에 바로 적용할 수 있을 것입니다.

## 문제가 생겼을 때

사용 중 문제가 발생하면 아래 내용을 확인해 보세요.

**URL 형식 오류가 표시됩니다**

주소가 `https://weflab.com/page/`로 시작하지 않는 경우에 발생합니다. 위플랩 페이지에 다시 접속해서 주소창의 URL을 새로 복사한 뒤 붙여넣어 보세요.

**화면이 비어 있습니다**

페이지 로드가 차단되었거나 네트워크에 문제가 있을 수 있습니다. 먼저 일반 브라우저에서 해당 URL이 정상적으로 열리는지 확인해 주세요.

**오버레이 창이 움직이지 않습니다**

고정 모드가 켜져 있는 상태입니다. 고정 모드를 끄면 오버레이를 다시 이동하거나 크기를 조절할 수 있습니다.
